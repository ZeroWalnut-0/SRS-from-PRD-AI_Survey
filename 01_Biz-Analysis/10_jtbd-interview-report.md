# 🃏 10. 핵심 JTBD 요약 카드 (Job-to-be-Done Summary Cards)

본 문서는 비즈니스 파이프라인을 관통하는 **3계층 코어 타겟**이 실제 우리 플랫폼의 '옵션(파싱/데이터맵/라우팅)'을 **"고용(Hiring)"**하고 수작업을 **"해고(Firing)"**하게 되는 생생한 심리와 동인(Dynamics)을 요약한 카드 체계입니다. 
탁상공론에 그치지 않고, "어떤 상황에서 이들의 지갑(또는 트래픽)이 열리는가?"에 대한 본질적인 답을 담았습니다.

---

## 🃏 Card 1. [Base 타겟] 홍일반 (대학생 / 소상공인 무과금 유저)
**"이름 치고, 나이 치고, 객관식 선지 타이핑하기 귀찮아 죽겠다. 그냥 내 문서 알아서 폼으로 띄워!"**

| 구분 (Field) | 상세 내용 (Details & Dynamics) |
| :--- | :--- |
| **Persona / Segment** | **Base (트래픽 화약고)** – 대학생 과제, 동아리 임원, 가벼운 의견 수집용 유저 |
| **Situation (상황)** | **모바일로 빨리 투표/조사를 걷어야 하는데, 이미 작성해 둔 워드/한글 기안의 질문을 폼 빌더에 일일이 쳐서 옮기기 귀찮을 때** |
| **Job Statement** | 툴 가입도 귀찮고 조작도 모르겠으니, **그냥 내 화면에 파일을 던지는 즉시 10초 만에 남들에게 뿌릴 수 있는 모바일 폼 창을 만들어 내고, 돈은 한 푼도 내지 않길 원한다.** 문서가 없으면 AI 챗봇에 '이런 조사 하고 싶어'라고 말해 설문 구조를 받고 대화하며 수정·확정하고 싶다. |
| **Desired Outcome** | **타이핑 시간 제로화. 복붙 피로도 삭제. 즉각적인 카톡 폼 배포 달성.** |
| **4 Forces (동인)** | **[Push/고통]** 구글 폼에 들어가서 질문/선지를 복붙하는 미칠듯한 귀찮음. 설문 방법론을 몰라 문항 구성 자체가 막막함.<br>**[Pull/매력]** "HWPX 파일 드래그 한 번 하니까 10초 만에 폼이 짜잔!" 또는 "AI 챗봇이 설문 구조를 추천해주니 방법론 몰라도 전문가 수준 설문 완성!"<br>**[Habit/관성]** 어차피 무료니까 그냥 다시 구글 폼 쓸까 하는 귀차니즘.<br>**[Anxiety/걱정]** 혹시 무료라고 하면서 나중에 결제 창 뜬는 건 아니겠지? |
| **Switch Triggers & Barriers** | **Trigger:** 조별 과제 마감이 1시간 남아서 1분 1초가 급박할 때.<br>**Barrier:** 플랫폼 화면 하단에 너무 크게 찍히는 '무료 워터마크'가 조금 거슬림. |
| **Evidence (가설 인용구)** | *"구글 폼 좋죠. 근데 문항이 20개가 넘어가면 그거 일일이 치다가 날밤 샙니다. 그냥 파일 갖다 대면 바로 화면 띄워주는 마법 같은 거 없나요? 돈 안 내는 대신 밑에 **[AI 무인 폼 변환] 광고 링크 워터마크** 붙는 거요? 아휴 상관없어요. 제 시간 아끼는 게 최고죠."* |

## 🃏 Card 2. [Core 타겟] 최실무 (기획/행정/마케팅 대리)
**"대행사 쓸 돈은 없고, 내가 엑셀 열고 데이터맵 수치 조작하다가는 오늘 퇴근 못 한다"**

| 구분 (Field) | 상세 내용 (Details & Dynamics) |
| :--- | :--- |
| **Persona / Segment** | **Core (캐시카우/현금창출)** – 기업 내 대행사(에이전시) 없이 조사를 돌려야 하는 일반 실무진 |
| **Situation (상황)** | **조사가 끝나고 걷힌 데이터들을 SPSS 같은 전문 통계 툴에 넣거나, 상부에 깔끔하게 분석표로 정리해서 올려야 할 때** |
| **Job Statement** | 구글 폼이 뼉어낸 외계어 엑셀을 일일이 숫자 코딩하는 수작업을 집어치우고, **대행사(엠브레인 등)에게 수백만 원 외주를 준 것처럼 완벽하게 각 잡힌 통계용 '데이터맵' ZIP 팩과 '서술형 결과 보고서 초안'을 직납받아 야근을 파괴하고 싶다.** 기안 문서를 올리면 AI 주치의가 오류를 잡아주길 원한다. |
| **Desired Outcome** | **데이터맵 작성 야근 시간 10시간 → 다운로드 3초. 외주 비용 0원.** |
| **4 Forces (동인)** | **[Push/고통]** 응답 1만 개 떨어졌는데, Q1=응답1, Q2=응답2 매핑하다 눈 빠질 것 같은 육체적 고통. 매크로 응답 걸러내는 노가다.<br>**[Pull/매력]** "AI가 불성실 응답은 다 걸러주고, 만 원만 내면 5종 엑셀 압축팩에 서술형 리포트 초안까지 쏟아진다고!?"<br>**[Habit/관성]** 그냥 내가 이번만 눈 딱 감고 수작업으로 엑셀 코딩표 짜맞출까.<br>**[Anxiety/걱정]** AI가 만들어준 데이터맵이 문항 번호랑 꼬이게 매핑된 건 아니겠지? |
| **Switch Triggers & Barriers** | **Trigger:** 결재/마감 날 엑셀 수식이 엉켜서 멘탈이 붕괴될 때 당장의 법인 카드 소액 결제로 위기 탈출.<br>**Barrier:** 회사 내부에서 외부 툴 소액 결제를 막는 부서 지침. |
| **Evidence (가설 인용구)** | *"폼 뿌리는 건 일도 아니죠. 문제는 수집된 거 깔끔하게 SPSS '데이터맵 명세서'로 엑셀 수기 코딩하고 할당표 짜는 건데 이거 누가 합니까? 결국 막내가 밤새워요. 근데 솔루션이 문항 원본 옆에 변수명 다 표기해주고, AI가 불성실 응답도 실시간으로 쳐내주고, 끝날 때 서술형 요약 리포트까지 한 번에 정리해 뱉어준다고요? 미쳤네요. 당장 부서 카드로 긁겠습니다."* |

## 🃏 Card 3. [Top VIP 타겟] 유팀장 (연구펌/조사 에이전시)
**"알바생 감시 비용이랑 패널 연동 외주 개발비 때문에 남는 마진이 없다"**

| 구분 (Field) | 상세 내용 (Details & Dynamics) |
| :--- | :--- |
| **Persona / Segment** | **Top VIP (장기 라이선스 락인)** – 리서치 펌, 여론조사 대행사 팀장급 이상 |
| **Situation (상황)** | **10만 명 규모의 외부 패널 조사가 떨어져 성/연령별 할당(Quota)이 터지는 걸 막아야 하고, 패널 회사로 성공/탈락 링크를 분기해 쏴줘야 할 때** |
| **Job Statement** | 폼 세팅 원가 파괴는 기본 베이스고, 조사 도중 터져 나오는 **'동적 할당(Quota) 엑셀 통제'와 '외부망 3단 라우팅(리다이렉트)'을 외주 개발이나 주말 알바생 없이 순수 노코드(No-code) 환경으로 끝내버리고 싶다.** |
| **Desired Outcome** | **연구 펌 내부 폼 세팅 원가, 주말 쿼터 모니터링 알바 비용, 라우팅 스크립트 개발 외주비용 전면 100% 삭제 (영업 이익률 한계 돌파).** |
| **4 Forces (동인)** | **[Push/고통]** 조사 하나 딸 때마다 인건비와 스크립트 알바비 나가고 남는 게 없음.<br>**[Pull/매력]** "이 SaaS는 뒷단에 우리 같은 기관만 쓰라고 [동적 교차 쿼터 서버]랑 [리다이렉트 입력란]이 뚫려있네?"<br>**[Habit/관성]** 그냥 쓰던 사내 구형 빌더나 계속 쓰지 뭐.<br>**[Anxiety/걱정]** 회사 메인 인프라를 여기 라이선스로 옮겼다가 이 회사 서버 터지면 우리 쪽 배상금 감당 어떡하지? |
| **Switch Triggers & Barriers** | **Trigger:** 패널사 연동 스크립트 에러가 나서 수백만 원 물어주고 주말 내내 카운팅 다시 해야 할 때 개강렬한 빡침.<br>**Barrier:** 연 단위 수천만 원에 달하는 전사 도입 엔터프라이즈 라이선스 비용 및 경영진 승인 절차. |
| **Evidence (가설 인용구)** | *"우리도 구글 폼 편한 거 다 압니다. 근데 그걸로 사업 장사가 됩니까? 우리는 외부 엠브레인이나 한국리서치 같은 패널사랑 라우팅 핑백 쏠줘야 하고, X축 Y축 실시간 할당(Quota) 블로킹을 걸어야 돈을 멡니다. 그거 하려고 개발 인건비가 몇천 나갔어요. 오픈서베이나 틸리언 프로도 써봤는데 쿼터 동적 제어나 패널 라우팅은 안 되더라고요. 근데 이 웹에서 엑셀 표 하나 올리면 쿼터제어가 바로 뚫리고 3단 링크가 뽑혀요? 연간 계약서 가져와요. 당장 전사 시스템 갈아치웁니다."* |
